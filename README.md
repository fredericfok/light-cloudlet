Welcome to your Node.js project on Cloud9 IDE!

This is the LightStream project , a NodeJs-based project 

The intent of this project is to : 
- expose REST services to benefit from Connected Light Management, leveraging SWAGGER
- expose OPC UA services to integrate with industrial supervision system
- faciliate deployment and hosting in cloud environment 
- leverage socket.io to deliver websocket-based communication with endpoints, servers, client apps

Roadmap: 
- mobility-related services 

## Running the server

1) Open `server.js` and start the app by clicking on the "Run" button in the top menu.

2) Alternatively you can launch the app from the Terminal:

    $ node server.js

Once the server is running, open the project in the shape of 'https://projectname-username.c9.io/'. As you enter your name, watch the Users list (on the left) update. Once you press Enter or Send, the message is shared with all connected clients.
